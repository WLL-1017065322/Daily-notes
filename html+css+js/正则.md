# 字符/ the/gi

`/` 是这个正则表达式的头部

`the` 是我们想要匹配的模式

`/` 是这个正则表达式的尾部

`g` 代表着 `global`(全局)，意味着返回所有的匹配而不仅仅是第一个。

`i` 代表着忽略大小写，意思是当我们寻找匹配的字符串的时候忽略掉字母的大小写。

# 数字 `/\d/g`。

在选择器后面添加一个加号标记(`+`)，例如：`/\d+/g`，它允许这个正则表达式匹配一个或更多数字。
